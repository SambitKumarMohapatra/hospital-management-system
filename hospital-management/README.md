# 🏥 Hospital Management System

A complete Hospital Management System built using **Java, Spring Boot, MySQL, HTML, CSS & JavaScript**.  
This project streamlines hospital operations with appointment booking, billing, prescriptions, and admin dashboard features.

---

## 🚀 Features

### 🔹 1. RESTful API with CRUD Operations
- Doctors, Patients, Departments, Appointments, Bills, Prescriptions
- Clean layered architecture (Controller → Service → Repository)
- Proper entity relationships using JPA

### 🔹 2. Appointment Management
- Book, update, cancel appointments
- Real-time doctor–department mapping
- Appointment status tracking:
    - Pending
    - Confirmed
    - Cancelled
    - Completed

### 🔹 3. Billing System
- Auto bill generation linked with appointment
- Charge calculation & tracking

### 🔹 4. Admin Dashboard
- Session-based login authentication
- Manage doctors, departments, patients
- Bill generation
- Appointment cancellation
- Prescription creation

### 🔹 5. Responsive Frontend
- HTML5, CSS3, JavaScript
- Dynamic form validation
- Real-time API integration

---

## 🛠️ Technologies Used
- **Java 21**
- **Spring Boot 3**
- **Spring Data JPA**
- **MySQL**
- **Maven**
- **HTML5, CSS3, JavaScript**
